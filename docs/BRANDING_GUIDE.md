## Typography

### Font Hierarchy

| Level | Font Family | Size | Weight | Usage |
|-------|------------|------|--------|-------|
| **H1** | Kaushan Script | 3rem | 600 | App title, hero headings |
| **H2** | Atkinson Hyperlegible | 2rem | 700 | Section headings |
| **H3** | Atkinson Hyperlegible | 1.5rem | 700 | Subsection headings |
| **H4** | Atkinson Hyperlegible | 1.25rem | 600 | Card headings |
| **Body** | Atkinson Hyperlegible | 1rem | 400 | Body text, paragraphs |
| **Small** | Atkinson Hyperlegible | 0.875rem | 400 | Secondary text, captions |

### Font Stack

**Primary Font (Body & H3-H4):** Atkinson Hyperlegible - Highly readable, professional, optimized for both screen and print

```css
font-family: "Atkinson Hyperlegible", "Lexend", "Inter", sans-serif;
```

**Display Font (H1-H2 & App Title):** Kaushan Script - a stylish hand-lettered display font that lends personality and warmth to our app while still remaining legible when styled with stronger strokes and spacing.

```css
font-family: 'Kaushan Script', 'Open Sans', cursive;
```

<div style="margin: 1rem 0; padding: 1rem; background: #e3f2fd; border-left: 4px solid #1976d2; border-radius: 4px;">
  <strong>Why Kaushan Script?</strong>
  <ul style="margin-top: 0.5rem; line-height: 1.8;">
    <li>‚úÖ <strong>Distinctive and friendly</strong> - Kaushan Script sets a unique tone without becoming unprofessional</li>
    <li>‚úÖ <strong>Customizable intensity</strong> - We increase perceived weight using text-stroke and shadow so it reads confidently on UI elements and headings</li>
    <li>‚úÖ <strong>Pairs well with Open Sans</strong> - Hand-lettered display + neutral body creates a pleasant contrast</li>
    <li>‚úÖ <strong>Improved readability</strong> - We provide additional spacing (kerning) and stroke for better clarity on small and large screens</li>
    <li>‚ùå <strong>Rejected alternatives:</strong>
      <ul style="margin-left: 1.5rem; margin-top: 0.25rem;">
        <li>Dosis - Too geometric/modern</li>
        <li>Cabin - Slightly less distinctive than Kaushan Script, though still professional</li>
        <li>Pacifico/Permanent Marker - Too casual for financial/sales interface</li>
      </ul>
    </li>
  </ul>
</div>
